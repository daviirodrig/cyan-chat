Info

Emote Sync is experimental. It can be laggy when there are many emotes being spammed at once. Even so it is the recommended default to have it on. Its purpose is to keep all emotes in sync with each other. If you have performance issues then you can use this to disable it.

Cyan Chat

Setup

Cyan Chat is an overlay that allows you to show your Twitch chat (and optionally Youtube chat) on screen with OBS, XSplit, and any other streaming software that supports browser sources. It supports your BetterTTV, FrankerFaceZ and 7TV emotes, always at the best available quality. You have many options to customize your chat, like enabling a smooth animation for new messages, or fading old ones after some time. If you have a chat full of !gamble addicts, you can choose to hide bots and commands messages. It also comes with many fonts and styling options that can be combined as desired. Custom fonts can either be installed on your system or be the name of any Google font. The "Center" style is not compatible with the animate option and will always have an animation.

Commands

Cyan Chat comes with a few mod only commands that you can use in chat.

  • !chat refresh - Refreshes the emotes in chat.
  • !chat reload - Reloads the overlay.
  • !chat rickroll - Rickrolls the chat.
  • !chat tts - Lets you send a tts message (Also can specify a voice with -v {voice}).
